# 📧 Email Extractor Automation Tool - CodeAlpha Internship Task

A professional Python automation tool that extracts email addresses from a text file using Regular Expressions (Regex).

This project was created as part of the CodeAlpha Python Programming Internship.

---

## 🚀 Features

- Extract emails automatically from text files
- Regex-based email detection
- Remove duplicate email addresses
- Display total unique email count
- Save extracted emails to output file
- Error handling for missing files
- Replay system
- Professional terminal interface

---

## 🛠 Technologies Used

- Python
- VS Code
- Regex (re module)

---

## 📚 Python Concepts Used

- Regular Expressions (Regex)
- loops
- conditional statements
- file handling
- exception handling
- user input/output

---

## ▶️ How to Run the Project

1. Open project folder in VS Code
2. Open terminal
3. Run the following command:

```bash
python email_extractor.py
```
